apis-servicios
==============

Apis basadas en cURL y PHP para los servicios estatales.

Servicios incluidos:
- Consulta saldo bip.
- Ubicación de buses y consulta ubicación próxima de paraderos transantiago (experimental)

Servicios por agregar:
- Farmacias de turno.
- Restaurantes (Sernatur).
- Bencinas en línea.
