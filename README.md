apis-servicios
==============

Apis basadas en cURL y PHP para los servicios estatales.

Servicios incluidos:
- Consulta saldo bip.
- Ubicación de buses y consulta ubicación próxima de paraderos transantiago (siempre en beta!)
- Restaurantes (Sernatur).
- Farmacias de turno.

Servicios por agregar:
- Bencinas en línea.
